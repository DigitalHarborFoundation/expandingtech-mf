Documenting with Google Drive

Documenting with Google Drive

In the previous Google Drive lesson, you practiced working with your personal Google Drive. This lesson focuses on how to use Google Drive to document the projects that you complete throughout Maker Foundations. In addition to your personal Google Drive, you’re going to be working with a Course Google Drive Folder that has already been shared with you.

Goals:

  • Understand what should go in the Course Drive
  • Set up your folder within the Course Drive

Personal Drive and the Course Drive

Throughout Maker Foundations, you’re going to be using the Course Drive for documentation and your personal Google Drive for storage. The distinction is that the folder that you set up within the Course Drive is used entirely for documenting your projects whereas your personal drive should be used to store any files that you’re actively working on, any resources that you find and want to save, and really anything else that you want to keep. Think of your personal drive as an extra, shared hard drive where you can keep works in progress that you can access from any internet connected computer.

The program staff will periodically be reminding you to always upload files to your personal drive, especially in the Graphic Design module. When you’re working at DHF you won’t always be able to get the same laptop each day, so it’s important to build the habit of uploading your files to your drive at the end of each session. This ensures that you won’t lose any work and project files!

Course Google Drive

The documentation that you do for each module’s mini-project will be uploaded to the Maker Foundations Course Google Drive. You’ll be creating and sharing your Maker’s Reflections and media inside

Drive Layout

Inside the Youth Documentation folder on the Drive, you’ll see that there are sub-folders for both the MW and TuTh Maker Foundations cohorts. Click on your cohort and then you’ll see a folder with your name. This is your personal folder! As an activity for this lesson, you’ll be setting up folders for each of the Maker Foundations modules.

Activity: Adding the Course Drive Folder

The goal of this activity is to add the shared course drive folder to your personal Google Drive. The Course Drive has been shared with you already, but you need to add it to your Drive so that you can setup the module folders.

Steps:

  1. Login to with your DHF email address.
  2. Navigate to Google Drive. Since you’re already logged in with your email, you’ll be taken directly to your personal drive.
  3. Find the Shared with me option in the left sidebar and click it. It becomes bold once you select it.
  4. Find the folder labeled 7 – Youth Documentation and right click. Select the Add to My Drive option from the menu.
  5. Select My Drive from the left sidebar and open up the course drive folder.
  6. Locate (and click to open) your cohort folder and find (and open) the folder with your name on it. In this example, Steph’s sample folder is selected:
  7. Open your folder!

Activity: Setting Up Your Folders

After you’ve added the course drive to your personal drive, you can start setting up your personal folder. The goal of this activity is for you to set up your personal drive folder inside the drive folder for your Maker Foundations cohort.

Steps:

  1. Make sure that you’re inside your named folder in the Course Drive.
  2. Click the blue NEW button on the left of the Drive window:
  3. Select Folder from the drop-down menu:
  4. Type the folder name in the window and then click the Create button:
  5. Create a folder for each of the 6 Maker Foundations modules. Here is an example where Steph has created the Graphic Design module folder:

    • Graphic Design
    • Game Development
    • 3D Printing
    • Circuits
    • Web Development
    • Capstone Project
  6. Inside each module folder create a Media folder as well. This is where you’ll store any photos and videos that you upload.

Pro-Tip: Make sure that you go back to your name folder each time you create a new folder or else you’ll make all the folders inside the Graphic Design folder!

The reason that you’re creating all of these folders is that you’re going to be uploading all of the project documentation for each module into the module folder. This will keep all of your work organized when it is time to review your progress with the program staff!
